# 项目概述 python4

本项目是一个基于FastAPI的博客系统，采用Python进行开发，使用MySQL作为数据库存储，通过SQLAlchemy实现对象关系映射（ORM），Elasticsearch提供全文搜索功能。前端部分，博客界面采用BootstrapVue，管理界面使用ElementUI，整体框架为Vue。

## 技术栈

- **后端**
  - Python
  - FastAPI
  - MySQL
  - SQLAlchemy
  - Elasticsearch

- **前端**
  - Vue
  - BootstrapVue（博客界面）
  - ElementUI（管理界面）

## 功能模块

- 用户管理
  - 用户注册
  - 用户登录
  - 用户资料管理
- 博客文章管理
  - 发布文章
  - 编辑文章
  - 删除文章
- 评论管理
  - 发表评论
  - 回复评论
  - 管理评论
- 搜索功能
  - 文章标题、内容搜索

## 系统角色

- 访客
- 注册用户
- 管理员

## 运行环境

- Python环境：推荐使用Python 3.6及以上版本
- 数据库：MySQL 5.7及以上版本
- 搜索引擎：Elasticsearch 7.x及以上版本

## 部署步骤

1. 安装Python环境
2. 安装并配置MySQL数据库
3. 安装并配置Elasticsearch搜索引擎
4. 克隆项目代码，安装依赖
5. 运行数据库迁移
6. 启动FastAPI服务
7. 构建并部署前端

- 基于FastAPI，提供高性能的API服务
